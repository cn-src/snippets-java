# Snippets Java
> 一些零散的Java代码片段，集成 jar 后可直接使用

* jOOQ Condition 动态创建支持
* jOOQ Geometry 几何类型支持
* jOOQ 额外函数支持
* Spring Data JDBC & jOOQ 扩展支持
* Spring Data & Eclipse Collections 扩展支持
* Kryo & Eclipse Collections 序列化支持
* 自定义 Spring 事务注解支持
* 完善的异常处理机制，消息国际化，OpenApi文档异常响应自动生成
* 请求/响应 详细日志支持
* 日期填充时间的转换支持
* Tree UI 数据结构支持
* 其它
